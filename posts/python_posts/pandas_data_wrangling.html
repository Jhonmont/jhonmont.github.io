def apply_function(dataframe,col):

    for i in col:
        dataframe[i] = dataframe[i].apply(lambda x: x/10)
    return dataframe

25. pivot_table()

  • Esta función se utiliza para crear una tabla dinámica a partir de un DataFrame. Una tabla dinámica es un resumen de datos agrupados por una o más variables, y es útil para la exploración y el análisis de datos a través de una función de agregación.
In [42]:
def pivot_table_function(dataframe,indx,col,values,agg_funct):

    dataframe = round(dataframe.pivot_table(index=indx, columns=col, values=values,

    return dataframe
